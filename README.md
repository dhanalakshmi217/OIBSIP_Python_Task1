# Voice Assistant

A simple Voice Assistant project in Python that can perform tasks like telling the time, weather updates, news, opening websites, and more using voice commands.

## Features

- Listen and respond to voice commands
- Tell current time and date
- Provide latest news headlines
- Fetch weather updates
- Open websites (e.g., YouTube, Google)
- Basic conversational responses

## Requirements

- Python 3.x
- Python packages:
  - `pyttsx3`
  - `speech_recognition`
  - `datetime`
  - `webbrowser`
  - `requests`
  - `json`

Install required packages using:

```bash
pip install pyttsx3 speechrecognition requests
